Example

If you type a 1 command it may response immediately.

Type the 'Sai [message]' this will be reply your message, it is fun!

Commands :

  • Help : it shows the all command list.

  • Info : it shows the info list.

  • Cosplay : it send a cosplay pictures.

  • Meme : it send a meme pictures.

  • Loli : it send a anime/loli pictures.

  • Milf : it send a 18+ pictures.

  • Uwu : it send a random tiktok video(shoti).

  • Wiki : it send the info of word you type.

  • Bible : it send a bible verses.

  • Catfact : it send a cat pictures and cat fact.

  • Dogfact : it send a dog pictures and dog fact.

  • Qtt : it will be send a random pickup lines.

  • Binary : it is converting a plain text into binary.

  • Repeat : it will be echo your message.

  • Uid : it send a user you mentioned id.

  • Unsent : it is unsent her own message.

  • Groups : it is shows the group list.

  • Fact : it send a random facts.

  • Lyrics : it send a song lyrics.

  • Ask : it is answering your questions.

  • Baybayin : it send a unidentified code.

  • Morse : it send a morse code.

Features :

  • Anti Unsent : if you unsend a message the bot will return it back.

  • Auto Reaction : if you type a sad or happy message, it will be send reaction

  • Solving Math : this bot is solving math problems using 'ask' command.

  • Answering Questions : this bot is answering questions using 'ask' command.
